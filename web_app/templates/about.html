Identifying Insincere Questions on Quora

The aim of this project is to identify insincere questions in the Quora questions dataset, in addition to creating an application (chrome add-on) that utilises our identification system to provide a sincerity score. The idea for the project comes from a Kaggle competition entitled Insincere Question Classification. It is a binary text classification problem which looks to identify insincere questions in a dataset provided by Quora, containing 1.3m training examples and 56k testing examples.

This website allows you to search through the questions in the test and training datasets and in case of the training dataset it also displays whether a question is sincere or not. Sincere questions have "target" value set to 0 while insincere questions have it set to 1. In addition to this website we have also created a chrome extension that is linked to our machine learning module and given a question, it gives you a prediction of whether the question is likely to be sincere or not. The extension and code for the machine learning module and this website can be found on our github page.

Below are some sample questions that can be used to test the chrome extension:

  • [Insincere] Do the Chinese eat all the primates?
  • [Insincere] Is Trump a penis?
  • [Sincere] Is Quora legit?
  • [Insincere] Why are all men bastards?
  • [Insincere] Was AIDS a punishment from God for gay people?
  • [Sincere] Why does velocity affect time?
  • [Sincere] How do I work in cyber security overseas?
